
# Hangman

A terminal based version of hangman.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hangman'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hangman

## Usage

run ruby lib/hangman.rb
